EVENT DASHBOARD

Day 1 Day 2

Mark Patterson

Hacker & Cybersecurity Trainer

Ethical Hacking 101 – Hands-on Penetration Testing

10:00 PM - 11:30 PM

Join Mark Patterson for an intensive hands-on workshop that will introduce you to the world of ethical hacking. This session covers the basics of penetration testing, including reconnaissance, scanning, vulnerability assessment, and exploitation. Participants will learn practical skills that can be immediately applied to identify and address security vulnerabilities in their organizations.

Dr. Sophia Reynolds

Chief Security Officer, CyberSafe Inc.

The Future of AI in Cybersecurity

1:00 AM - 3:00 AM

Dr. Sophia Reynolds explores the cutting-edge intersection of artificial intelligence and cybersecurity. This talk will delve into how AI is transforming the security landscape, from advanced threat detection to automated response systems. Learn about both the potential benefits and the unique challenges that AI brings to modern cybersecurity practices.

Raj Patel

Cloud Security Architect, FortiCloud

Securing Cloud Infrastructure

2:00 PM - 3:30 PM

Raj Patel presents a comprehensive overview of cloud security best practices. This session covers critical aspects of securing cloud infrastructure, including identity and access management, network security, data protection, and compliance frameworks. Attendees will gain practical insights for implementing robust security measures across various cloud platforms.

Emma Chen

Security Researcher, DefenseTech Labs

Zero-Day Vulnerabilities: Detection and Mitigation

4:00 PM - 5:30 PM

Emma Chen shares advanced techniques for identifying and addressing zero-day vulnerabilities before they can be exploited. This technical session will cover vulnerability research methodologies, exploit development analysis, and strategies for building resilient systems that can withstand previously unknown attack vectors.

Dr. James Wilson

Director of Cybersecurity, TechDefend

Building a Security-First Organization

6:00 PM - 7:30 PM

Dr. James Wilson discusses organizational approaches to embedding security into company culture. This leadership-focused session covers security governance frameworks, building effective security teams, managing security budgets, and creating security awareness programs that actually work across all levels of an organization.

Myth Busting

Myth vs Fact

  • Myth: Cybersecurity is only for large companies.
  • Fact: Every organization, regardless of size, is at risk.
  • Myth: Antivirus software is enough.
  • Fact: A layered security approach is essential.

Did You Know?

  • Over 40% of cyber attacks target small businesses.
  • Hackers can breach systems in under 5 minutes.
  • Continuous training and awareness are key to defense.
